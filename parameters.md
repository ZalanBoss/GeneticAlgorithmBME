# Parameters to set

* mutation function (2x)
* selection (2x)
* max acceleration (3x)
    * max angular velocity (1x)
* max velocity (3x)
    * max angular acceleration (1x)
* initial population (3x)
* map (3x)
    * fitness function (2x)
* mutation chance (3x)

# Parameter values

* '--mutationtype' {0,1}
* '--selection' {0,1}
* '--maxacc' {75, 250, 475}
* '--maxvel' {75, 150, 225}
* '--initial' {15, 50, 100}
* '--map' {0,1,2}
* '--fitness' {0,1}
* '--mutationchance' {0.000125, 0.0025, 0.05}
